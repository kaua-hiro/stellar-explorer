# 🪐 Stellar Explorer Mobile

Uma aplicação mobile de alta performance desenvolvida para exploração astronômica, focada em UX fluida e estruturação de dados dinâmica.

## 🚀 Tecnologias e Conceitos
* **React Native & Expo:** Desenvolvimento mobile cross-platform.
* **JavaScript (ES6+):** Lógica de estados e manipulação de fluxos de dados.
* **Componentização:** Arquitetura baseada em componentes reutilizáveis.
* **Dynamic Rendering:** Mapeamento de objetos complexos para interface nativa.

## 🧠 Engenharia do Projeto
O projeto utiliza um motor de renderização dinâmico que processa coleções de dados (Planetas e Galáxias) e gera interfaces em tempo real através de modais performáticos e listas otimizadas.
